## 3. 当前实现语义要点

### Outbox 状态机

```text
pending → leased → published
              ↓
      retry_scheduled → leased → published (循环)
              ↓
         dead_letter （达到 MAX_TENTATIVE=3）
```

### Delivery 状态机

```text
pending → sending → sent
              ↓
      retry_scheduled → sending → sent (循环)
              ↓
            failed （达到 MAX_DELIVERY_TENTATIVE=3）
              ↓
          unknown → sent (仅通过 reconcile)
```

### 重试策略

- 指数退避：base=10s, multiplier=3, max=3600s
- 尝试次数在 lease 时递增（attempt_count +1）
- 在 retry 时比较 attempt_count >= MAX_TENTATIVE 决定 dead_letter/failed

### Lease 语义

- 条件更新：UPDATE ... WHERE event_id=? AND status=? AND lease_version=?
- lease_version 每次递增，publish/retry 验证 owner+version
- 过期 Lease 通过 RecoveryService 回收（status = 'pending'）

### RecoveryService

- recover_outbox_leases：status='leased' + lease_expires_at < now → 'pending'
- recover_delivery_leases：status='sending' + lease_expires_at < now → 'pending'
- recover_stale_turns：status='running' + active_attempt_id → attempt=abandoned, turn=queued
- unknown Delivery 不受影响（保持待 reconcile）

## 4. 有意延期项

以下规范要求在实现中有意延期，原因如下：

| 规范 | 延期原因 |
|---|---|
| Turn 的 next_attempt_at 字段 | 自动重试逻辑尚未实现（需要 Dispatcher 支持 queued+next_attempt_at 过滤），当前通过显式 Command 触发重试 |
| Lease 超时精确配置 | 当前 lease_expires_at 设置为当前时间，后续应改为 now + TTL |
| Delivery 流式状态 | 流式投递（streaming/finalizing）需要真实 Channel 支持，当前状态机已准备但流程未实现 |
| Connector/主动推送 | 属于 P4+ 阶段，依赖当前基础 |
| 完整 Outbox Consumer 链 | 当前 Outbox 只完成发布，未实现 Event Consumer（消费 outbox → 触发 Delivery） |
| Turn expired 状态 | 过期扫描尚未实现，需要 Scheduler 组件 |
| Runtime/Python 3.12 datetime adapter 警告 | sqlite3 默认 datetime 适配器弃用警告，待 Python 3.14 方案稳定后统一修复 |

## 6. 已知风险

1. 当前使用 SQLite `datetime` 文本序列化，Python 3.12+ 的 `sqlite3` 模块有 DeprecationWarning（datetime 适配器），计划在 Python 3.14 方案稳定后修复。
2. 测试中 `max_retries_leads_to_failed` 手工将 status 设为 `pending` 模拟重试循环——生产环境中通过 RecoveryService 或 Lease 过期实现。
3. 未实现跨进程 Worker 的 Lease 心跳续期，当前依赖 RecoveryService 在启动时回收过期 Lease。
4. Dispatcher 的 `recover_stale_turns` 简化实现——当前将所有 running Turn 标记为 abandoned，未区分 Lease 有效。
