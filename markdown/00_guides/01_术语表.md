# 统一术语表

> **权威边界**：本文只统一名称；字段和状态以对应功能规范为准。

## 1. 交互与执行

| 术语 | 定义 | 禁止混用 |
|---|---|---|
| Principal | 系统识别并授权的主体 | 平台显示名称 |
| Endpoint | LangBot 转换后的稳定外部身份或投递端点引用 | SDK 临时对象 |
| Conversation | 平台对话容器和排序边界 | Session |
| Session | 由 Channel/Conversation 决定的短期上下文边界 | 跨 Channel 用户画像 |
| Message | 不可变的标准化消息事实 | 流式 Delta |
| Turn | 一次用户意图的逻辑生命周期 | 单次模型请求 |
| RunAttempt | Turn 的一次实际执行尝试 | 独立 Run 层 |
| Task | 可等待、重试、跨重启的后台工作 | Event Handler 调用 |
| TaskAttempt | Worker 对 Task 的一次执行占用 | Task 本身 |

## 2. 数据与异步

| 术语 | 定义 |
|---|---|
| Command | 请求系统改变状态的意图，必须有调用者和幂等语义 |
| Event | 已经发生、不可变的事实 |
| Inbox | Consumer 已接收/处理消息的幂等记录 |
| Outbox | 与业务状态同事务写入、事务外投递的意图 |
| Checkpoint | 恢复执行所需的最小确定状态 |
| Receipt | 外部副作用结果或可对账证据 |
| Projection | 可从事实重建的查询视图 |

## 3. Agent 与认知

| 术语 | 定义 |
|---|---|
| Agent Loop | 模型输出、Tool 执行和继续推理组成的有界循环 |
| Context Snapshot | 某次 Attempt 实际使用且不可变的上下文选择结果 |
| Memory Candidate | 尚未被接受为长期事实的提议 |
| MemoryItem | 带来源、Scope、置信度和生命周期的长期认知事实 |
| Preference | MemoryItem 的当前偏好视图 |
| Goal | MemoryItem 的活动目标视图 |
| Drift | 仅在资源空闲时执行、默认只读的低优先级维护工作 |

## 4. 能力与安全

| 术语 | 定义 |
|---|---|
| Capability | Tool、Skill、Connector、Provider 等可注册能力 |
| Provider | 某个稳定端口的完整实现 |
| Middleware | 固定调用点上的校验或变换，不拥有业务状态机 |
| Policy Gate | 确定性授权决策点 |
| Approval | 对固定参数快照和动作哈希的持久授权对象 |
| Sandbox | 限制文件、网络、进程和资源的执行隔离边界 |

## 5. 状态词约定

- `queued`：可以被调度但尚未取得执行权；
- `running`：当前有且仅有一个有效执行权；
- `waiting_user`：等待持久 Approval/用户输入，不持有 Lease；
- `waiting_external`：等待可描述的外部条件；
- `unknown`：外部副作用结果不确定，必须先 reconcile；
- `failed`：当前逻辑对象失败，是否重试由显式策略决定；
- `abandoned`：旧 Attempt 已丧失执行权，其结果不得提交。
